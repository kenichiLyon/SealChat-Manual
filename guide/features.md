# 功能指南

本指南按功能拆分为更细的模块，便于查找与系统学习。

## 功能目录

- [新手引导](/guide/features/onboarding)：欢迎页、教程中心与进度保存
- [消息与聊天](/guide/features/messaging)：发送、编辑、悄悄话、归档与表情
- [骰子与跑团工具](/guide/features/dice)：骰子表达式、宏、托盘与隐骰
- [身份卡与角色](/guide/features/identities)：角色身份与权限管理
- [附件、图库与音频](/guide/features/assets)：文件上传、图库与音频库
- [搜索与导出](/guide/features/search-export)：历史搜索、导出与筛选
- [社交与世界](/guide/features/social)：好友、世界大厅、频道归档与在线状态
- [协作工具](/guide/features/tools)：iForm、术语库与便签
- [显示与通知设置](/guide/features/settings)：布局、主题与通知管理
- [Webhook 集成](/developer/webhook)：消息同步与外部对接
- [SealDice 对接](/developer/sealdice)：跑团机器人联动

## 使用建议

- 新用户先阅读新手引导与消息/骰子模块
- 管理员关注协作工具与集成扩展
- 运维人员可结合管理员指南与配置参考
