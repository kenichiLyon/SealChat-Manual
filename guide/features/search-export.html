搜索与导出 ​

消息搜索 ​

基础搜索 ​

  1. 点击频道顶部的搜索图标
  2. 输入关键词
  3. 按 Enter 搜索

高级搜索 ​

支持多种搜索条件：

条件语法说明
关键词魔法搜索包含"魔法"的消息
发送者from:用户名搜索某用户的消息
日期after:2024-01-01搜索某日期之后的消息
日期before:2024-12-31搜索某日期之前的消息
频道in:频道名搜索特定频道

组合搜索：

text
from:艾露莎 魔法 after:2024-01-01
1

搜索结果 ​

  • 结果按时间排序
  • 显示匹配消息的上下文
  • 点击可跳转到原消息位置

筛选面板 ​

搜索面板提供更细粒度筛选：

  • 发言人筛选（支持多选）
  • 时间范围筛选
  • 是否包含场外消息

消息导出 ​

导出功能可以将聊天记录保存为文件，方便整理和分享。

导出入口 ​

  1. 进入频道设置
  2. 选择"导出消息"
  3. 或使用命令 /export

导出格式 ​

格式说明适用场景
纯文本.txt 格式快速备份、二次处理
HTML网页格式，保留样式分享、存档、打印
海豹染色器BBCode/Docx跑团记录整理与排版

导出选项 ​

选项说明
时间范围选择要导出的消息时间段
包含 OOC是否包含 OOC 消息
包含归档是否包含已归档消息
隐藏时间导出时移除时间戳
合并消息合并同一用户的连续消息

导出流程 ​

  1. 选择导出参数
  2. 点击"开始导出"
  3. 等待后台处理
  4. 处理完成后下载文件

后台处理

大量消息的导出可能需要一些时间。导出任务在后台执行，不影响正常使用。导出完成后会收到通知。

HTML 导出查看器 ​

导出的 HTML 文件包含独立的查看器：

  • 无需联网即可查看
  • 保留原始格式和样式
  • 支持搜索和导航

海豹染色器上传（可选） ​

如管理员启用云端上传，导出为海豹染色器格式时可自动上传生成 BBCode/Docx。
