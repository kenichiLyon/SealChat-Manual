页面导航

协作工具 ​

iForm 表单 ​

iForm 是嵌入频道的交互表单，用于收集信息。

查看表单 ​

  • 表单显示在频道顶部或消息流中
  • 点击可以填写和提交

创建表单（管理员） ​

  1. 进入频道设置
  2. 选择"iForm 管理"
  3. 点击"创建表单"
  4. 添加字段：
    • 文本框
    • 单选
    • 多选
    • 日期
    • 数字

展示方式 ​

  • 面板嵌入：固定在频道顶部
  • 抽屉模式：侧边栏打开
  • 浮窗模式：可拖拽到任意位置
  • 支持弹出新窗口查看

表单用途 ​

  • 角色信息收集
  • 投票调查
  • 签到统计
  • 规则确认

术语与关键词 ​

术语功能可对频道消息进行关键词高亮与解释。

创建术语（管理员） ​

  1. 进入世界设置 → 术语管理
  2. 新建关键词与描述
  3. 可设置分类、别名与匹配模式（普通/正则）

展示模式 ​

  • 标准：高亮并展示说明
  • 精简：仅高亮关键词
  • 继承：跟随世界默认设置

导入与导出 ​

  • 支持 JSON 与 CSV 导入
  • 可按分类导出备份

便签功能 ​

便签是频道内的轻量协作工具，可用于公告与流程辅助。

查看便签 ​

  • 便签显示在频道中，可置顶或浮窗
  • 可根据权限控制可见范围

编辑便签（管理员） ​

  1. 点击便签区域的编辑按钮
  2. 输入或修改内容
  3. 支持富文本格式
  4. 保存

便签类型 ​

  • 文字便签：发布公告、规则、链接
  • 清单便签：可勾选的任务列表
  • 计数器：资源值追踪
  • 滑条/计时器/时钟：进度与回合管理

便签用途 ​

  • 频道规则说明
  • 重要公告
  • 常用信息汇总
  • 链接收集
